Image processing

Introduction to image and video processing using software and hardware

Posted by Jose Manuel Ramirez R. on June 16, 2020

Motivation

This project will give us a brief introduction to visual computing, it's main features, some implementations and (hopefully) we'll understand image and video processing basics.

Goals

As well as the visual computing class, we want to understand the digital images manipulation, creation and production, these processes will allow us to create a modern visual application, using a computer architecture. In this project, we want to study:

  • Relations: The relation between computer and human vision
  • Inputs: Geometric models, data, images and gestures
  • Stages: Which and how to code each stage.

Methodology

Use the processing 3 API and execute the following "framework":

  • Set image and video variables
  • read folder data and assign to variables
  • Go through each pixel and get the actual color
  • Process the actual pixel and assign to a new variable
  • Show original and modified image/video
