Visual illusions

For this project, the following illusions will be implemented:
  • Hermann grid (interactive)
  • Dot waves
  • Penrose triangle
Hermann grid (interactive)

For this process, we'll create a 10 x 10 grid, with black/white boxes that changes when the mouse hovers.

for (var i = 1; i <= 100; i++) { document.write('<div class="square"></div>'); } #Grid { width: 500px; height: 500px; margin-left: 50%; position: relative; left: -250px; } .square { width: 40px; height: 40px; margin: 5px; background-color: #000000; float: left; } #Grid:hover { background-color: #000000; } #Grid:hover .square { background-color: #ffffff; } h1 { text-align: center; margin-top: 100px; } You can see the demo here
Dot waves

For this process, we rotate 2 united arcs, forming a circle with a different arc size (cyan color), and making all the "circles" rotate with a small angle, forming a wave pattern by only rotating on it's center.

for (let i = 50; i <= width - 50; i +=50) { for (let j=50; j <=height - 50; j +=50) { push(); translate(i, j); rotate(HALF_PI + i + angle); fill("green"); stroke("blue"); strokeWeight(3.5); arc(0, 0, 30, 30, 0, PI + j); pop(); push(); translate(i, j); rotate(QUARTER_PI + j + angle); fill("green"); strokeWeight(3.5); stroke("cyan"); arc(0, 0, 30, 30, PI + i, 0); pop(); angle +=0.0004; } } You can see the demo here
